Welcome

BBgoggles is an open source project to provide a working BlackBerry version of the popular Google Goggles application for iOS/Android.

Using BBgoggles is as easy as 1, 2, 3! Just start the application, take a picture with Camera, upload the image to the server and then retieve a list of matches. You can then visit a webpage containing information about that result.

Visit the downloads page to download and try out the Application.
